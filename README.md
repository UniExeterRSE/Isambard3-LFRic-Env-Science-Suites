# Isambard3 LFRic Environment + Science Suites

## Archive Notice

This repository is now an archive and is no longer the source of truth for
LFRic Spack packages or CSC environments. It has been superseded by the Met
Office repositories:

- [MetOffice/mo-spack-packages](https://github.com/MetOffice/mo-spack-packages)
- [MetOffice/csc-environments](https://github.com/MetOffice/csc-environments)

The historical code, environment definitions, and suite handover material remain
available on the `main` branch for reference, but new work should use the
repositories listed above.
